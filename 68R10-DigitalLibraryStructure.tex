A {\em digital library structure} is a triplet $(G,L,F)$, where $G=(V,E)$ is a directed graph with a vertex set 
$V$ and edge set $E$, $L$ is a set of label values, and $F$ is labeling function 
$F:(V\cup E)\rightarrow L$.
