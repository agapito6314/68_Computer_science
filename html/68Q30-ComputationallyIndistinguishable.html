computationally indistinguishable


If {Dn}n∈ℕ and {En}n∈ℕ are distribution ensembles (on Ω) then we say they are computationally indistinguishable if for any probabilistic, polynomial time algorithm A and any polynomal function f there is some m such that for all n>m:

|ProbA(Dn)=ProbA(En)|<1p⁢(n)

where ProbA⁡(Dn) is the probability that A accepts x where x is chosen according to the distribution Dn.
